# Image Search with PixaBay API

### `Built With`

- React
- Material UI
- Axios
- PixaBay API

### `Installation`

```properties
# clone the project
git clone https://github.com/jungwon-oh87/pixabay_image_search.git

# navigate to the directory
cd pixabay_image_search

# install packages
npm install or yarn

#start the ap
npm start or yarn start
```
